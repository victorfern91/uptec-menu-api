# uptec-menu-api

### API methods
- `/menu/`, returns all week menu
![week example](http://i.imgur.com/4LCTiI5.png)
- `/menu/:day`, returns the menu for a selected day (could use `today`)
![day menu](http://i.imgur.com/XZfe7vb.png)

### Requirements

- `wget` installed

### installed
```
npm install
```

### Run
```
npm start
```
